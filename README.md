
# Tic Tac Toe
> Just a simple TicTacToe game that uses Machine Learning.

## Setup
The model is already trained, so you only need to run it.

```bash
# run python server 
python ./tictactoe.py

# run golang server
go run tictactoe.go

# build and run golang server
go build tictactoe.go
./tictactoe
```
